## Dispensables > Data Class

- Aucune méthode
- Bcp d'autres classes en dépendent
    - ViewModel ou DTO ?

**Corriger**

- Move / Extract Method 
- Hide Method / Remove Setting Method
- Encapsulate Field / Collection

Note: 

Data classes doivent se responsabiliser pour justifier leur existence.
